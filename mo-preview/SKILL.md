# mo-preview

マークダウンファイルを `mo` コマンドでプレビューする。

## 実行方法

対象がファイルの場合、Bash ツールで以下を実行する：

```bash
mo path/to/file.md
```

対象がディレクトリの場合、そのディレクトリ内の `.md` ファイルを全て開く：

```bash
# ディレクトリ内のすべての .md ファイルに対して mo を実行
find path/to/dir -name "*.md" | xargs -I{} mo {}
```

## トリガー条件

1. **明示的に開くよう指示された場合**
   - 「moで開いて」「プレビューして」「開いて」などと言われ、対象がマークダウンファイルまたはディレクトリの場合

2. **マークダウンファイルの作成完了後**
   - ユーザーが「マークダウンで書き出して」「.mdファイルで作成して」などと指示し、Claudeがファイルの書き出しを完了した直後
   - ファイルを Write ツールで保存したら、続けて `mo` で開く
   - ディレクトリ単位で指示された場合（例:「このディレクトリのmdファイルを書き出して」）も同様

## 注意

- 複数のマークダウンファイルを作成・対象にした場合は、それぞれに対して `mo` を実行する
- `mo` はホットリロードに対応しているため、既に開いているファイルを編集・追記した場合は再度 `mo` を実行しなくてよい
- 「マークダウン形式で答えて」のような、ファイルを作らない指示には使わない
